# SQL Database Project

This project demonstrates the design and implementation of a relational database for a music school.

## Database Structure

The database contains four main tables:

- **teachers** – information about music teachers
- **students** – information about students
- **courses** – music courses offered by teachers
- **enrollments** – links students to courses

## Features

- Creation of relational database schema
- Use of primary keys and foreign keys
- Sample data insertion
- SQL queries using JOIN to retrieve information

## Example Queries

- Retrieve all students
- Retrieve all teachers
- Show courses with their teachers
- Display student enrollments

## Technologies

- SQL
- Relational database design
